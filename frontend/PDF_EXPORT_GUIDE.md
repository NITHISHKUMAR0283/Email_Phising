# Professional PDF Export Feature - Implementation Guide

## 📋 Overview

This document explains the **Professional PDF Export Feature** implemented for the PhishGuard AI email security application. The feature generates a "Forensic Threat Report" PDF that looks like an official audit report or invoice.

## 🚀 Installation

### Step 1: Install Required Dependencies

Run this command in the `frontend` directory:

```bash
cd frontend
npm install jspdf jspdf-autotable
```

Or with yarn:
```bash
yarn add jspdf jspdf-autotable
```

### Step 2: Verify Installation

Check your `frontend/package.json` to confirm both packages are listed under `dependencies`:

```json
{
  "dependencies": {
    "jspdf": "^2.5.1",
    "jspdf-autotable": "^3.5.31",
    ...
  }
}
```

## 🏗️ Architecture & Design

### Components

The implementation consists of two main files:

#### 1. **frontend/src/utils/pdfExport.ts** (Utility Module)
- Contains `exportForensicPDF(email, securityDNA?)` function
- Generates professional PDF using jsPDF + jsPDF-AutoTable
- Exported as a reusable utility

#### 2. **frontend/src/components/EmailDetailPage.tsx** (Consumer Component)
- Imports `exportForensicPDF` from the utility
- Calls it when the "📄 Export Report" button is clicked
- Passes current email data and security analysis metrics

### Data Flow

```
User clicks "📄 Export Report" button
         ↓
generateForensicReport() function executes
         ↓
exportForensicPDF(email, securityDNA) is called
         ↓
PDF is generated with professional formatting
         ↓
Browser downloads PDF file automatically
         ↓
Privacy event is logged for audit trail
```

## 💻 Code Usage Example

### In EmailDetailPage.tsx

```typescript
import { exportForensicPDF } from '../utils/pdfExport';

export default function EmailDetailPage({ email, onBack }: EmailDetailPageProps) {
  // ... component logic ...

  const generateForensicReport = () => {
    try {
      const securityDNAData = {
        headerAuth: 65,
        linkAnalysis: 58,
        contentAnalysis: 55,
        spfDkim: 72,
        dmarc: 43,
      };

      exportForensicPDF(email, securityDNAData);
      
      logPrivacyEvent('Exported Forensic PDF Report', { 
        reportType: 'PDF', 
        emailSubject: email.subject
      });
    } catch (error) {
      console.error('PDF Export Error:', error);
      alert('Failed to generate PDF report.');
    }
  };

  return (
    <button onClick={generateForensicReport}>
      📄 Export Report
    </button>
  );
}
```

## 📊 PDF Export Function Signature

```typescript
export function exportForensicPDF(
  email: EmailData,
  securityDNA?: SecurityDNAData
): void
```

### Parameters

**`email: EmailData`** (Required)
```typescript
{
  id: string;              // Unique email identifier
  subject: string;         // Email subject line
  sender: string;          // From address
  risk_score: string;      // Risk level (Safe/Suspicious/Critical)
  final_score: number;     // Risk percentage (0-1)
  timestamp: string;       // ISO timestamp
  highlight?: {
    urls: string[];        // Suspicious URLs
    phrases: string[];     // Phishing phrases
  };
  ai_analysis?: {
    explanation: string;
    red_flags: string[];
    suspicious_phrases?: string[];
    recommendation?: string;
  };
}
```

**`securityDNA?: SecurityDNAData`** (Optional)
```typescript
{
  headerAuth: number;      // SPF/DKIM/DMARC score (0-100)
  linkAnalysis: number;    // URL analysis score
  contentAnalysis: number; // Content phishing score
  spfDkim: number;        // Email signing score
  dmarc: number;          // DMARC validation score
}
```

### Return Value
- `void` - The function directly triggers a browser download
- PDF is saved as: `PhishGuard_Report_[SUBJECT]_[TIMESTAMP].pdf`

## 🐛 Troubleshooting

### Issue: "Module not found: jspdf"

**Solution**: Run `npm install jspdf jspdf-autotable` in the frontend directory

```bash
cd frontend && npm install jspdf jspdf-autotable
```

### Issue: PDF shows but looks broken

**Solution**: Restart the development server:
```bash
npm run dev
```

### Issue: Table columns are misaligned

**Solution**: Check that the `SecurityDNAData` object has valid percentage values (0-100)

### Issue: Special characters in subject causing issues

**Solution**: The code automatically sanitizes filenames. Check browser console for detailed error message.
